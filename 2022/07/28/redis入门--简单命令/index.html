键命令：

命令 描述
DEL key 删除键
DUMP key 序列化键对应的值
EXISTS key 判断键是否存在
EXPIRE key seconds 设置键的过期时间
EXPIREAT key timestamp 设置键在具体的一个时间过期，参数是时间戳的形式
PEXPIRE key milliseconds 设置键的过期时间，参数以毫秒为单位
PEXPIREAT key milliseconds-timestamp 设置键在具体的一个时间过期，参数是毫秒时间戳的形式
KEYS pattern 匹配符合的KEY
MOVE key db 将键移到其它的数据库
PERSIST key 将KEY的过期时间去掉，永久保存
PTTL key 返回Key的有效时间，以毫秒为单位
TTL key 同上，以秒为单位
RANDOMKEY 随机返回一个Key
RENAME key newkey 重命名，如果newkey已经存在，则会覆盖
RENAMENX key newkey 重命名，不过只有newkey不存在时才会成功
[SCAN cursor MATCH pattern] [COUNT count] 遍历键，每次返回一个
TYPE key 返回Key的类型

String(字符串)：

  • string是redis中最基本的数据类型，一个key对应一个value。
  • string 类型是二进制安全的。意思是它可以存储任何数据，比如jpg图片和序列化后的对象。
  • 最多可以存储512MB
